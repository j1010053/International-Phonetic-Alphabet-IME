# Changelog

## 0.1.1 — POC 第一輪回饋修正

**依 Windows POC(V1–V6)測試結果調整。**

### 輸入行為
- **numpad 綁為選字鍵**(KP_1–9 選第 1–9 候選),補上主鍵盤數字列不能選字之缺口;
  主鍵盤數字列送字面數字,供 `|` / `^` 碼使用。〔待再測〕
- **標點優先序修正(V3-a)**:`.`、`|`、`\` 三者補「指向自己、最高權重」候選,
  字面標點排第一。
- **`\t.` 尾端 `.` 延遲修正(V3-b)**:自訂 punctuator 僅保留 `, ? !` 即時上屏,
  `.` `|` `\` 交由 speller 於組字時取用;新增 recognizer 規則確保 `\` 起首之
  Praat 碼整段落入 speller。〔待再測〕

### 聲調 / 上標(依需求變更)
- **取消 `td`(組合調符)**:組合調符改用 X-SAMPA `_T/_H/_M/_L/_B`,足夠直覺。
- **調値字母改 `|` + 數字**:1~3 位、每位 1–5,逐位映射 ˩˨˧˦˥
  (`|2`→˨、`|13`→˩˧、`|232`→˨˧˨)。共 155 列。取代原 `tl`。
- **新增 `^` + 數字 = Unicode 上標數字** ¹²³⁴⁵…:1~2 位允許 0–9、3 位限 1–5
  (`^35`→³⁵、`^214`→²¹⁴)。共 235 列。

### 顯示
- 新增 `weasel.custom.yaml`:候選視窗字型設 Charis SIL(接收端字型弱時之緩解;
  V5 已確認輸入法輸出正確,顯示問題純在接收端字型)。

### 資料
- master:173 → 549 列。dict 條目:497 → 850。內部/主流撞名:0。

### 已知待驗

  重新吃掉,改用 Space/方向鍵選字或改綁定。
- V3-b 之 punctuator/recognizer 調整需再測確認 `\t.` 即時顯示、`.` 不誤上屏。
